## Write a Program to add two numbers:

``` c
// Program

#include <stdio.h>
void main()
{
    // define var
    int a = 5, b = 6;

    // print sum of a & b
    printf("Then sum of two number is: %d", a + b);
}

```

### Output: <br/>
Then sum of two number is: 10
