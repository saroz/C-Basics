## Write a Program to display character.

``` c
// Program

#include <stdio.h>
void main()
{
    // define var
    char a;

    // ask user to enter character
    printf("Enter a character: ");
    scanf("%c", &a);

    // print output
    printf("The character you entered is: %c", a);
}

```

### Output: <br/> 
Enter a character: r <br/>
The character you entered is: r
