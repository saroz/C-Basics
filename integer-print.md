## Write a Program to print integer.

``` c
// Program

#include <stdio.h>
void main()
{
    // define var
    int a;

    // ask user to enter value of a
    printf("Enter the value of a: ");
    scanf("%d", &a);

    // print output
    printf("You enter the number %d", a);
}

```

### Output: <br/> 
Enter the value of a: 5 <br/>
You entered the number 5.
